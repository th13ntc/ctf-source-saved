Welcome to the Alex Fan Club API homepage

About

This API can be used by anyone to query the (wow!) achievements of Alex Fan.

Features

This API uses XML to query requests from a secure SQL database containing Alex Fan Club's list of Alex Fan's achievements. Additionally, requests expire after about a minute because that is cool!

Usage

To query the API, you have to make a POST request to {{ apiurl }} with two parameters. The time parameter contains a base64 urlsafe representation of the epoch time. For example, the current epoch time is {{ epochtime }}, and the urlsafe base64 encoded representation is {{ basesftime }}. The search parameter contains the search query for the API enclosed in <search></search>. The API returns the results of the search separated by newlines.

An example of a python script that queries the string "cool" to the API is the following

            import requests
import base64
import time

api = "{{ apiurl }}"

req_time = int(time.time())

data = {
    "time": base64.urlsafe_b64encode(req_time.to_bytes((req_time.bit_length() + 7) // 8, byteorder="big")),
    "search": "<search>cool</search>"
}

x = requests.post(api, data=data)
print(x.text)

The above code snippet will print the following

            Too cool for #include <bits/stdc++.h>
Is super cool
Has a cool profile picture
